Dashboard Comercial – Pet Shop

Contexto do Projeto

Este é um projeto desenvolvido com dados fictícios de um Pet Shop também fictício, criado com finalidade educacional e para demonstrar possibilidades analíticas utilizando o Power BI.

O cenário simula um pet shop localizado no bairro do Paar, em Belém do Pará, onde os dados são gerados a partir de cadastros realizados no momento das compras. Essas informações permitem explorar padrões de consumo e comportamento dos clientes.

Estrutura de Dados

O modelo foi estruturado a partir das seguintes tabelas principais:

  • Clientes: informações como gênero, estado civil e dados cadastrais.
  • Vendas: registros de cada compra realizada, incluindo produtos e quantidades.
  • Produtos: catálogo com categorias e características dos itens comercializados.

Objetivo Analítico

A partir desses dados, foi desenvolvido um dashboard comercial com o objetivo de:

  • Analisar o desempenho geral do pet shop
  • Identificar produtos e serviços mais vendidos
  • Entender padrões de comportamento dos clientes
  • Apoiar decisões estratégicas para aumento de vendas

Resultado

O projeto demonstra como a modelagem adequada dos dados e a construção de indicadores estratégicos podem transformar dados operacionais em informações relevantes para gestão e tomada de decisão.

Problema de Negócio

O desafio na criação de um dashboard é transformar dados brutos em insights. Neste caso, a ideia é criar um painel que oferecesse uma visão clara e abrangente das vendas, dos produtos mais rentáveis e do comportamento dos clientes.

Modelagem de Dados

Estrutura baseada em modelo dimensional com fato de vendas e dimensões de Produto, Categoria, Cliente e Calendário.

Principais KPIs

  • Faturamento Total
  • Quantidade Vendida
  • Ticket Médio
  • Faturamento por Bairro
  • Crescimento Mensal

Stack Utilizada

Power BI • DAX • Power Query • Modelagem Dimensional
